# Required Documents

This register lists **representative** lifecycle artifacts. Exact titles and locations follow program standards (USSM tiers, repository layout, and change control).

## Normative standards

| Document | Notes |
| --- | --- |
| `USSM — Unified Software Standards Manual v1.0.md` | Active controlled standards spine for SDLC documentation tiers, ISO/IEC 12207 and IEEE 29148 alignment, traceability, compliance checklists, and annex guidance. The Master Lifecycle phase files implement USSM operationally; Appendix A and standalone `*-001` files provide local controlled templates. |

| CYBERCUBE Name Registry | `../CYBERCUBE standards/[1]-CYBERCUBE-Name-Registry.md`; authoritative lookup for registered Namespace A artifact codes, Namespace B CC-PID entity codes, Namespace G product/module IDs, Namespace M module records, UUID-to-CC-PID mappings, and project registry entries. |
